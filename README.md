# metakurly-springboot-project

## 메타넷 SpringBoot 미니 프로젝트 공간입니다.

참여자: 강혜경, 손정아, 이현주, 신예빈 <br/>
기간: 2023.03.24 ~ 2023.03.28 <br/>

<역할> <br/>
PM: 신예빈 <br/>
DB Master: 이현주 <br/>
UI Master: 손정아 <br/>
Git Master: 강혜경 <br/>

<기능 담당> <br/>
신예빈 - 회원  <br/>
손정아 - 상품 및 리뷰 조회 <br/>
이현주 - 주문 및 결제 <br/>
강혜경 - 장바구니 <br/>
